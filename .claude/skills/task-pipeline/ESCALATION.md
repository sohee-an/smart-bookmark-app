# 에스컬레이션 기준

## 5회 실패 시 처리

테스트 5회 연속 실패 시 아래를 순서대로 실행:

1. 구현 중단
2. Notion 세부 태스크 상태 → `검수요청`
3. 큰 태스크 상태 → `검수요청`
4. Notion 댓글에 아래 내용 자동 기록:

```
[검수요청] 5회 실패로 사람 확인이 필요합니다.

실패한 테스트:
- [테스트명]: [실패 이유]

시도한 방법:
1. [시도 1]
2. [시도 2]
...

마지막 에러:
[에러 메시지]
```

## 검수요청 이후

사람이 댓글로 지시 후 상태 전환:

| 사람의 판단       | 다음 상태  | AI 행동                         |
| ----------------- | ---------- | ------------------------------- |
| 방향 수정 지시    | 방향성     | 태스크 내용 수정 후 재검수 대기 |
| 직접 수정 후 계속 | 개발중     | 수정된 내용 기반으로 재시작     |
| 태스크 폐기       | 완료(폐기) | 종료                            |

## 실패 횟수 초기화 조건

- 사람이 방향 수정 지시 후 재시작 → 횟수 0으로 초기화
- 새로운 태스크 시작 → 횟수 0으로 초기화
