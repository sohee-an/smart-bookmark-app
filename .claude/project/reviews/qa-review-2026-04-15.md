## Critical (즉시 수정)

### [C-1] ErrorBoundary 전혀 없음

**파일**: `apps/web/src/app/layout.tsx`, 모든 페이지 컴포넌트
JS 에러 하나에 전체 흰 화면. `GlobalErrorBoundary`를 `layout.tsx`의 `<QueryProvider>` 안에 배치, 각 페이지도 독립 ErrorBoundary로 격리.

### [C-2] BookmarkDetailPanel — 저장 실패 시 사용자 피드백 없음

**파일**: `apps/web/src/entities/bookmark/ui/BookmarkDetailPanel.tsx:64`
오프라인 상태에서 저장 클릭 시 스피너만 멈추고 아무 안내 없음. catch 블록에 `toast.show` 추가.

### [C-3] AddBookmarkOverlay — 애니메이션 타이밍 이슈

**파일**: `apps/web/src/features/bookmark/ui/AddBookmarkOverlay.tsx:116`
`if (!isOpen) return null` 패턴으로 DOM 제거/마운트 시 저사양 기기에서 진입 애니메이션 미실행. opacity/scale/pointer-events 전환 방식으로 교체 권장.

### [C-4] CollectionDetailContent — `window.confirm` → iOS WebView에서 항상 false

**파일**: `apps/web/src/app/(main)/collections/[id]/CollectionDetailContent.tsx:66`
PWA 모드에서 삭제 버튼 동작 안 함. `BookmarkDetailPanel`의 인라인 확인 UI 패턴으로 교체.
