## High (출시 전 수정)

### 1. 로그아웃 중복 클릭 방어 없음

- **파일**: `apps/web/src/components/layout/Header.tsx:151`
- **Playwright**: ❌ 실패로 실제 버그 확인됨
- **재현**: 로그아웃 버튼 빠르게 2회 클릭 → `supabase.auth.signOut()` 중복 호출
- **영향**: 전체 로그인 사용자
- **수정**:

  ```ts
  const [isLoggingOut, setIsLoggingOut] = useState(false);

  const handleLogout = async () => {
    if (isLoggingOut) return;
    setIsLoggingOut(true);
    storage.cookie.remove("is_guest");
    await supabase.auth.signOut();
    router.push("/landing");
  };
  ```

  버튼에 `disabled={isLoggingOut}` 추가

### 2. 모바일 검색 버튼 터치 타겟 WCAG 미달

- **파일**: `apps/web/src/components/layout/Header.tsx:224`
- **재현**: 모바일(375px)에서 검색 아이콘 버튼 `h-9 w-9` = 36×36px
- **영향**: WCAG 2.5.5 기준 44×44px 미달 — 손가락이 두꺼운 사용자
- **수정**: `h-9 w-9` → `h-11 w-11` (44px)

### 3. 아바타 드롭다운 `group-hover:block` — transition 무효화

- **파일**: `apps/web/src/components/layout/Header.tsx:272`
- **Playwright**: ✅ 통과 (드롭다운 자체는 보이지만 animation은 작동 안 함)
- **재현**: `hidden → block` 전환은 CSS transition 적용 불가 — `animate-in fade-in zoom-in-95` 클래스가 실제로 작동하지 않음
- **영향**: 전체 로그인 사용자 — 드롭다운이 뚝 나타남
- **수정**:

  ```html
  <!-- 현재 -->
  <div class="hidden group-hover:block ...">
    <!-- 수정 -->
    <div
      class="pointer-events-none opacity-0 transition-all duration-200 group-hover:pointer-events-auto group-hover:opacity-100"
    ></div>
  </div>
  ```

### 4. 게스트 유저 `initialized=false` 구간 FOUC 잠재적 위험

- **파일**: `apps/web/src/components/layout/Header.tsx:157`
- **Playwright**: ✅ 통과 (현재 환경에선 재현 안 됨, 저사양 기기에서 가능)
- **재현**: `isGuest = initialized && !user && ...` — `initialized=false` 구간에서 `isGuest`는 무조건 `false`. `initialUser`가 null(게스트)이면 "로그인" 버튼이 순간 표시될 수 있음
- **수정**: `initialIsGuest` prop을 서버 layout에서 쿠키 읽어 내려주거나:
  ```ts
  // (main)/layout.tsx
  const isGuest = request.cookies.get("is_guest")?.value === "true";
  <Header initialUser={user} initialIsGuest={isGuest} />
  ```

## 잘된 부분

- **FOUC 방어 패턴 정확히 구현**: `currentUser = initialized ? user : (initialUser ?? null)` — 로그인 사용자의 깜빡임 방지 올바르게 적용됨. Playwright 테스트 통과로 실제 확인.
- **서버 컴포넌트에서 user fetch**: `layout.tsx`에서 `createSupabaseServerClient`로 서버에서 유저 가져와 `initialUser`로 내려주는 올바른 Next.js 패턴
- **모바일 검색 버튼 aria-label**: `aria-label="검색"` 명시되어 있음
