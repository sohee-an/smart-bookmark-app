# 📌 SmartMark — 제품 기획서

## 1. 제품 개요

URL을 저장하면 AI가 자동으로 요약, 태깅, 의미 기반 검색을 해주는 북마크 관리 서비스.

**핵심 차별점**

- 단순 키워드 검색이 아닌 **시맨틱 서치** — "React 최적화"를 검색하면 관련 의미를 가진 글도 함께 검색됨
- 브라우저에 쌓인 기존 북마크를 **AI가 자동으로 카테고리 분류**해서 정리해줌
- **크롬 익스텐션**으로 브라우징 중 원클릭 저장

---

## 2. 타겟 유저

- **1차**: 개발자 (기술 아티클, 문서 정리)
- **2차**: 지식 관리가 필요한 일반 사용자

**유저 페인포인트**

- 브라우저 북마크가 수백 개인데 어디 있는지 모름
- 나중에 보려고 저장했는데 다시 찾기가 어려움
- 키워드가 정확히 기억나지 않아도 검색하고 싶음

---

## 3. 플랫폼 구성 및 역할 분리

| 플랫폼            | 역할               | 주요 사용 맥락             |
| ----------------- | ------------------ | -------------------------- |
| **웹앱**          | 관리 / 검색 / 협업 | 앉아서 북마크 정리할 때    |
| **크롬 익스텐션** | 수집 / import      | 브라우징 중 바로 저장할 때 |

> 익스텐션은 "수집 도구", 웹은 "관리 도구"로 역할이 명확히 분리됨.
> 익스텐션에서 저장한 것은 웹에 단방향으로 전송됨 (양방향 동기화 없음).

---

## 4. 기능 목록

### 4-1. 웹앱

#### URL 저장

- URL 입력 → 자동 크롤링 (메타데이터, 본문 추출)
- 저장 즉시 카드 표시 (aiStatus: crawling → processing → completed)
- AI 자동 요약 생성 (Gemini API)
- AI 자동 태그 추출

#### 북마크 관리

- 북마크 카드 목록 보기
- 우측 슬라이드 상세 패널 (제목/태그 편집)
- 태그 필터링 (복수 선택, OR 조건)
- 읽음/삭제/수정

#### 검색

- 키워드 검색 — 제목/요약 텍스트 매칭
- 시맨틱 서치 — 임베딩 벡터 유사도 기반 의미 검색
  - 정확한 결과 (similarity ≥ 0.80)
  - 연관된 결과 (similarity < 0.80)

#### 인증

- 이메일/비밀번호 로그인
- Google OAuth
- 비회원 사용 가능 (localStorage, 5개 제한)
- 로그인 시 비회원 데이터 자동 이전

#### 협업 (컬렉션)

- 컬렉션(폴더) 생성 / 수정 / 삭제
- 북마크 컬렉션에 추가/제거
- 멤버 이메일 초대
- 멤버 역할 관리 (owner / editor / viewer)

---

### 4-2. 크롬 익스텐션

#### 원클릭 저장

- 현재 보고 있는 페이지 즉시 저장
- 로그인 연동 (웹앱과 동일 계정)

#### 브라우저 북마크 Import

- 크롬 북마크 전체 불러오기
- 폴더 구조 트리 형태로 미리보기
- 가져올 북마크 선택 (전체 선택 / 개별 선택)
- 선택한 북마크 웹앱으로 가져오기

#### AI 북마크 정리

- Import한 북마크들을 AI가 카테고리별로 자동 분류
- 중복 북마크 자동 감지 및 제거
- 정리 결과 미리보기 → 사용자 확인 → 브라우저 폴더에 적용
- **적용 흐름**: import → AI 분류 → 미리보기 → 확인 → 브라우저 폴더 업데이트

## 6. AI 기능 비용 정책

AI 북마크 정리는 북마크 수에 비례해서 API 비용이 발생함.

**1단계 (포트폴리오 MVP)**

- 계정당 무료 정리 횟수 제한 (예: 월 1회 or 100개까지)
- 한도 초과 시 안내 메시지 표시
- 결제 시스템 없음

**2단계 (실서비스 전환 시)**

- 유료 플랜 도입
- 무료 티어 한도 유지, 초과분 결제

## 7. 유저 플로우

### 신규 유저 (브라우저 북마크 정리)

```
익스텐션 설치
    ↓
로그인
    ↓
브라우저 북마크 import
    ↓
AI 자동 카테고리 분류 + 중복 제거
    ↓
정리 결과 미리보기
    ↓
확인 → 브라우저 폴더 업데이트
    ↓
웹앱에서 시맨틱 서치로 검색
```

### 일반 사용 (브라우징 중 저장)

```
브라우징 중 좋은 글 발견
    ↓
익스텐션 클릭 → 원클릭 저장
    ↓
웹앱에서 AI 요약/태그 확인
    ↓
시맨틱 서치로 나중에 다시 찾기
```

## 8. 검색 상세 기획

### URL 구조

```
/bookmarks                              # 전체 목록
/bookmarks?tag=React                    # 태그 단일 필터
/bookmarks?tag=React&tag=Next.js        # 태그 다중 필터 (OR)
/bookmarks?q=리액트+상태관리            # 키워드 검색
/bookmarks?q=리액트+상태관리&mode=semantic  # 시맨틱 검색
```

### 시맨틱 결과 분리

```
정확한 결과 (similarity ≥ 0.80)  ← 찾던 것과 거의 일치
연관된 결과 (similarity < 0.80)  ← 관련 있지만 다른 주제
```
